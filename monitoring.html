Technologies

CollectD

Daemon that periodically collects system and application performance metrics

It is capable of storing the values in a variety of ways: this implementation uses RRD files

CollectD Exporter

An exporter for collectd that transforms and exposes metrics for consumption by Prometheus

It leverages collectd's native 'Network' plugin to send metrics over HTTP POST

Prometheus

Leading open source monitoring tool

Dimensional data. Powerful queries. Efficient storage. Great visualisation. Precise alerting.

Grafana

Leading open source software for time series analytics

Integration with Prometheus through native adapter
